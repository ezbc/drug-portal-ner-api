Drug Adverse Event Named Entity Recognition

Description

Drug adverse events are reported as unstructured notes by physicians when a patient is experiencing complications from one or more drugs. The physician submits the report which ends up to the drug manufacturers. The drug manufacturers must then code the physician's report for side effects.

This demo uses an entity extraction model trained with the Python spacy library to reduce the time spent manufacturers code adverse event reports. For more details visit this blog post describing the app:

https://ezbc.me/2018/08/07/drug-adverse-event-entity-extraction
